Siopao's First Stretch is a simple, one-level platformer where the goal is to sling Siopao onto different platforms so he can reach the steamer basket.

The folders and files included in this compressed folder are the following:
- assets folder
--- siopao spritesheet.png
--- steamer.png
- CastilloDolinaEvangelista_FinalProject_COA.pdf
- entt.hpp
- Main.cpp
- README.txt

Compile and run Main.cpp as you would any C++/Raylib project.

On Mac:
clang++ -framework CoreVideo -framework IOKit -framework Cocoa -framework GLUT -framework OpenGL [full_path]/libraylib.a -std=c++17 [full_path]/Main.cpp -o [full_path]/executable && ./executable
